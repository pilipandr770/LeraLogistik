# Lera Logistics

**AI-керована платформа вантажних перевезень, інтегрована з біржею Lardi-Trans.**

Це MVP для експрес-перевірки гіпотези: чи може команда з однієї людини + 3-4 AI-агентів ефективно виконувати роботу традиційного диспетчерського відділу на українському логістичному ринку.

---

## TL;DR — за 5 хвилин до першого запуску

```bash
# 1. Клонувати репозиторій
git clone <your-repo-url> && cd lera-logistics

# 2. Підняти Postgres у Docker
docker compose up -d

# 3. Створити .env з ключами
cp .env.example .env
# — відредагуйте .env і вставте реальні LARDI_API_TOKEN і ANTHROPIC_API_KEY

# 4. Встановити залежності (рекомендуємо uv; або звичайний pip)
pip install -e ".[dev]"

# 5. Застосувати міграції (створить таблиці в БД)
alembic revision --autogenerate -m "initial schema"
alembic upgrade head

# 6. Запустити
uvicorn app.main:app --reload

# → відкрийте http://localhost:8000
```

## Технологічний стек

- **Python 3.12+**
- **FastAPI** — веб-фреймворк
- **SQLAlchemy 2.0** (async) + **asyncpg** — ORM та драйвер
- **Alembic** — міграції
- **Pydantic 2** / **pydantic-settings** — валідація + конфіг
- **Anthropic SDK** — Claude API для AI-агентів
- **httpx** — HTTP-клієнт з підтримкою async
- **tenacity** — retry для зовнішніх API
- **Jinja2** + **HTMX** + **Tailwind CSS** + **Alpine.js** — UI
- **pytest** (+ pytest-asyncio) — тести
- **ruff** — лінтер і форматтер

Повний список — в `pyproject.toml`.

## Деплой на Render.com

Повний гайд: [`docs/DEPLOY.md`](docs/DEPLOY.md). Коротко:

1. У Render створіть Web Service → підключіть цей репозиторій.
2. Render прочитає `render.yaml` і налаштує білд + health-check автоматично.
3. У секретах сервісу додайте: `DATABASE_URL` (лінк до вашого Postgres), `APP_SECRET_KEY`, `LARDI_API_TOKEN`, `ANTHROPIC_API_KEY`.
4. Перший деплой автоматично застосує міграції і підніме сервіс.

Тарифний план Starter ($7/міс) — достатньо для MVP і тестування SWOT.

## Безпека і відповідальність

- **API-токени — через `.env`**, ніколи не в коді і не в Git.
- **Feature-прапори для агентів у `.env`**: на старті `AGENT_NEGOTIATOR_AUTO_SEND=false` та `AGENT_AUTO_ACCEPT_DEALS=false`. AI пропонує — людина підтверджує.
- AI Matcher — **безпечний**: він лише пише рядки в таблицю `matches`, не взаємодіє із зовнішнім світом.
- **Важливо**: кожне повідомлення, яке AI відправляє від вашого імені, це юридична дія. Не вмикайте автовідправку, поки не матимете 100+ підтверджених людиною угод.
